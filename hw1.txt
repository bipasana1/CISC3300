What happens from the time a user puts a URL into their browser to when they are able to view a webpage? - Briefly define ISP, URL, DNS, web server.
When a user puts a URL (Uniform Resource Locator) into their browser it specifies the location of the resource on the Internet if it is connected by ISP (Internet Service Provider); a service that gives the user access to the Internet. Then, the browser will take us to DNS (Domain Name System), a human-readable domain name that converts into an IP address the computer can understand. User requests are then sent to the web server hosting the webpage, every URL has a web server. Lastly, the user can finally view the webpage.
